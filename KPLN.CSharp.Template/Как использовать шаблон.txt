1. [Создание/корректировка шаблона]
Для того, чтобы создать шаблон, нужно проект из данной папки экспортировать как шаблон. Для этого - открываешь его в Visual Studio, идешь по пути:
Project -> Export Template...


2. [Подгрузка для локальных пользователей]
Для того, чтобы создавать новые проекты на основе шаблона для плагинов KPLN, необходимо zip-файл KPLN_CSharp_Template.zip скопировать локально себе в папку:
%USERPROFILE%\Documents\Visual Studio 2022\My Exported Templates

!!!Не забудь проверить версию Visual Studio
